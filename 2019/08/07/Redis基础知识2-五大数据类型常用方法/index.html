列表(List)常用方法

List注意是事项：

  • 它是一个字符串链表，left、right都可以插入添加。
  • 如果键不存在，创建新的列表；如果键已存在，增加内容。
  • 如果值全部移除，对应的键也就消失了。
  • 链表的操作无论是头和尾效率都是极高的，但假如是对中间元素进行操作，效率就相对较低。
